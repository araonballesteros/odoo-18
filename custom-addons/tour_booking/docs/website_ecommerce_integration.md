# Integración futura con Website y Ecommerce

Este módulo se centra actualmente en la gestión básica de reservas de paquetes turísticos. En una fase posterior se prevé añadir funciones para integrar el catálogo y la venta en línea a través de los módulos Website y Ecommerce.

## Pasos previstos
1. Crear vistas web públicas para mostrar los paquetes turísticos.
2. Habilitar el flujo de compra desde el sitio web utilizando el módulo `website_sale`.
3. Sincronizar las reservas realizadas en línea con este módulo.

Esta documentación se irá actualizando conforme avance el desarrollo.
